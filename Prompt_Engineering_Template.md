# Prompt Engineering Template

## System Prompt
Defines assistant behavior.

Example:
You are Basirah automation assistant helping network engineers with monitoring and automation tasks.

## User Prompt Template
User intent + context.

Example:
User question: {question}
Environment: {env}
Logs: {logs}

## Guardrails
- No hallucinated commands
- Professional tone
- Domain restricted answers

## Advanced Patterns
- Few-shot examples
- Tool descriptions
- RAG context injection
